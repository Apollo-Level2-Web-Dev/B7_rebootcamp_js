# 🧩 Task: Build a Simple CLI Program

## 📌 Objective

Create a command-line interface (CLI) program that takes user input and displays basic information.

## 🛠️ Requirements

### 1. Input

- Take **name** and **age** from the terminal.

### 2. Output

Your program should print:

- A greeting message using the provided name
  - Example: `Hello, John!`
- Whether the user is:
  - **Adult** (age ≥ 18)
  - **Minor** (age < 18)

### 3. Bonus ⭐

- Handle invalid inputs gracefully:
  - Non-numeric age
  - Empty name
  - Negative age
- Display helpful error messages instead of crashing

## 💡 Example

### Input
```

Name: Alice
Age: 22

```

### Output
```

Hello, Alice!
You are an Adult.

```

## 🚀 Notes
- Use Javascript.
- Keep the code simple and readable.
- Bonus points for clean error handling and user-friendly messages.
